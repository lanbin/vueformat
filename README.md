# vueformat
Sublime Text 3 插件  
Vue文件格式化工具 ( Vue file format plugin for Sublime Text 3)


# Usage

### on MacOS

``` shell
$ cd "~/Library/Application Support/Sublime Text 3/Packages/"
$ git clone https://github.com/lanbin/vueformat.git
```

### Hot Key

> command + shift + h


### Dependencies

1. npm
2. git
